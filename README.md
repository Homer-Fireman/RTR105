termināļa komandas:
- whoami

- ls - kas ir pieejams directorija

- cd - change directori  
-  ~ = tilde = home directory

- nano - texta editors
- cat -> izlasa failu  
- rm -> dzēst failu vai ar rmdir -> tukšu directory
